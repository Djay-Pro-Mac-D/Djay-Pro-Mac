## Overview

Djay Pro Professional Suite is a complete DJ platform with hardware support, video mixing, and performance recording for professional sets.

Hardware integration supports Pioneer DJ, Reloop, Numark, and other DJ controllers with pre-mapped configurations and MIDI learning for custom mapping.

Video mixing combines visual content with audio mixing for VJ-style performances with video transitions and effects.

Performance recording captures the complete DJ set as a high-quality audio file for promotional mixes and archival.

Djay Pro Professional Suite is the DJ platform for performers who want AI-powered mixing with the hardware support, video capability, and streaming access that modern DJ performance demands.

## Performance & Integration

Djay Pro is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Djay Pro processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
